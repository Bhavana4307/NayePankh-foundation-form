
# NayePankh AI Career Mentor

## Overview

NayePankh AI Career Mentor is an AI-inspired web application designed to help students explore career paths in technology. The application provides personalized recommendations based on a student's interests and presents them through an interactive AI chat interface.

## Features

* AI-style chat interface
* Personalized career recommendations
* Dark mode user interface
* Responsive design
* Interactive animations
* Career guidance across multiple technology domains
